[native-in-practice](//xudafeng.github.io/slide/archives/native-in-practice)

## Coverage

```bash
$ ./gradlew clean createDebugCoverageReport
```

## Linter

```bash
$ ./gradlew check
```

## DataHub

```bash
$ npm i macaca-datahub -g
$ datahub server -c ./macaca-datahub.config.js --verbose
```
